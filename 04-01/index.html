Example 4-1: Reuse the Same Values

Variables in Javascript do not have a type. Use var instead of float, int, double, long, char, String, Array, etc... read more 
var y = 60;    // use var instead of int
var d = 80;    // use var instead of int
ellipse(75, y, d, d);
ellipse(175, y, d, d);
ellipse(275, y, d, d);
