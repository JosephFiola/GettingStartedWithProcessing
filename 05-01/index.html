Example 5-1: The draw() Function

Open your browser's console window to see the output of the print() function.
This example is also using the text() function to print directly into the canvas. read more
mText = "I'm drawing";
	
print(mText);                        // print to console
print(frameCount);                   // print to console
	
text(mText, 20, 20, 170, 80);        // print into canvas
text(frameCount, 20, 40, 170, 180);  // print into canvas
